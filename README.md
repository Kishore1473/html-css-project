A modern and responsive personal portfolio website built using HTML and CSS.
This project showcases my skills, projects, and contact information in a clean and professional layout.

🚀 Features

- Modern responsive design
- Hero section with introduction
- About me section
- Projects showcase with cards
- Contact section
- Clean UI with hover effects
- Simple and lightweight

🛠️ Built With

- HTML5
- CSS3
- Flexbox
- Responsive design principles

📂 Project Structure

portfolio
│
├── index.html
└── style.css

💻 Getting Started

Follow these steps to run the project locally:

1. Clone the repository

git clone https://github.com/Kishore1473/html-css-project

2. Navigate to the project folder

cd portfolio

3. Open "index.html" in your browser.
